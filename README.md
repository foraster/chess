# TypeChess

A browser-based chess game built with React and TypeScript, featuring complete standard chess rules and an object-oriented architecture.

## Tech Stack

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS
- **Build Tooling:** Create React App

## Features

- **Full Chess Logic:** Move validation, check/checkmate/stalemate detection, castling, en passant, and pawn promotion.
- **Game Controls:** Rapid timer (10+0), undo/redo, pause, and full move history notation.
- **Interface:** Highlighted legal moves, check warnings, captured pieces panel, and light/dark theme.
- **Architecture:** Clean OOP domain model (`Board`, `Cell`, `Figure`) with board simulation for legal move validation.

## Getting Started

1. Install dependencies:  `npm install`

2. Start the development server: `npm start`
